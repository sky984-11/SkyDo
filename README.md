## 简介

一个高效简洁的待办事项程序，帮助您快速记录和完成重要的任务和计划。
持续迭代中...有bug欢迎反馈，反馈邮箱：1269861316@qq.com

## 技术栈

前端技术：Vue2 + vite作为前端框架，用于构建用户界面。采用tauri进行程序打包


## 优点

1. 界面干净，简洁，无任何广告
2. 体积小巧，不到5M
3. 自定义主题
4. 开机自启
5. 所有数据均存储在本地，避免安全隐患
