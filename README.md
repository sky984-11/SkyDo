## 简介

一个高效简洁的待办事项程序，帮助您快速记录和完成重要的任务和计划，并且可通过微信进行任务提醒。


## 优点

1. 界面干净，简洁，无任何广告
2. 可以使用微信接收任务提醒(因为目前用的微信测试号，所以暂时仅前100人可使用)
3. 自定义主题
4. 开机自启
5. 所有数据均存储在本地，避免安全隐患
